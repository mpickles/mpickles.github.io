Traveling Salesman Quantum Algorithm

Case Study | Python, Qiskit

After being inspired by this paper, I decided to spend time learning how the researchers Quantum Alogirthm worked and learn how to implement it myself. By encoding the cost (distance/time/fuel usage) of traveling between cities into the phase ofa qubit, we can create linear combinations of these phases and find the minimum. This is done by encoding all possible routes into eigenstates and then using a phase estimation alogirthm to get the binary output of the phase. This binary output is then converted to integers and an algorithm is used to find the minimum.

For this project, I decided to find the minimum travel time (cost) between San Diego, Los Angeles, Santa Barbara, and Palm Springs using the above algorithm. Then, to find the minimum cost, Durr and Hoyer's Quantum Algorithm for finding the minimum was implemented as to return the index of the minimum cost
