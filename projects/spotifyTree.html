Spotify L-System Trees

Web Application | React & Express.js

This application allows a Spotify user to create procedural generated trees using lindermeyer systems. The colors used for the trees are calculated taking the average tempo, energy, valence, and key values of the users recently played songs and mapping them to RGB values.
