ABOUT THIS PROJECT

These two projects were for an undergraduate linear/non-linear optimization class that focused on using pure linear algebra and calculus as opposed to built-in python packages to construct classification models and neural networks.

In the classification project, we built least squares based binary and multi-class classifiers to classify the MNIST handwritten digit dataset. The main focal point of the project was using Singular Value Decomposition to solve the Normal Equation, which is used in finding the solution to the least squares problem. The first classifier was a binary classifier where there could only be two labels. The second classifier extended the binary classifier to K classes, and for the third classifier we used designed pairwise Boolean classifiers for every pair of classes. Finally, we implemented Randomized Feature Based Least Squares Classifiers that used non-linear mappings to create a "feature space" that would be used to train the classifiers, and we compared the error rates.

In the Neural Network Project, we designed simple neural networks that used non-linear least squares to approximate non-linear functions. Using non-least squares algorithms, the network calculates the weights necessary to approximate the given non-linear functions. Specifically, we used the Levenberg-Marquardt algorithm to solve the non-linear least squares problem. To test the network, we randomly generated 500 different data-points and minimized the training loss with respect to the vector of weights that were used to parameterize the network.
