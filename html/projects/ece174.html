ABOUT THIS PROJECT

These two projects were part of my undergraduate linear/non-linear optimization class, where the focus was on using pure linear algebra and calculus instead of relying on built-in Python packages to construct classification models and neural networks.

In the first project, I focused on classification tasks by building least squares-based binary and multi-class classifiers aimed at classifying the MNIST handwritten digit dataset. A key highlight of this project was the utilization of Singular Value Decomposition (SVD) to solve the Normal Equation, a crucial step in finding solutions for the least squares problem. I designed three classifiers:

  • Binary Classifier: This classifier dealt with scenarios where there were only two possible labels
  • Multi-class Classifier: An extension of the binary classifier, this one accommodated K classes.
  • Pairwise Boolean Classifiers: For the third classifier, we crafted pairwise Boolean classifiers for every pair of classes.

I also implemented Randomized Feature-Based Least Squares Classifiers that used non-linear mappings to construct a 'feature space' used in training the classifiers and compared the error rates across these models.

In the Neural Network Project, the focus shifted to designing simple neural networks tailored to approximate non-linear functions. Using non-least squares algorithms, the network calculated the necessary weights for approximating these non-linear functions. Specifically, I employed the Levenberg-Marquardt algorithm to address the non-linear least squares problem. To put our network to the test, we generated 500 diverse data points and minimized training loss by adjusting the weight vector to parameterize the network.
