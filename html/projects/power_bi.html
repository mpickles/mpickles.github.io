Power BI

ABOUT THIS PROJECT

In this project, I loaded to data from previous running back analysis projects into Power bI to create visualizations of my data. The data contained seasonal statistics, weekly statistics, red zone statisitics, and snap-usage percentages of all fantasy relevent running backs from 2017-2023. Within Power BI, I created several visualizations:

  • Key Factors Analysis
  • Age vs Fantasy Points
  • Red Zone Usage
  • Snap Percentage
  • Running Back Stats vs Points
  • Efficiency
